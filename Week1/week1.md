## 📅 Day 1 - React Introduction & Setup

### Topics:
- React kya hai (beginner level)
- Node.js & npm install
- React app create (Vite)

### Tasks:
1. Node.js download aur install karo
2. npm verify karo (`npm --version`)
3. Vite se React app banao: `npm create vite@latest my-app -- --template react`
4. Dependencies install: `npm install`
5. Development server chalao: `npm run dev`
6. Browser mein app dekho

### Output:
✅ React development environment ready
